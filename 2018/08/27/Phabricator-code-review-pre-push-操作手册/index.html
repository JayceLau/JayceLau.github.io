概述

Phabricator支持两条不同的代码审查工作流，一条叫做“reivew”（审查），是在推送前进行的；另一条叫做“audit“（审计），是在推送后进行的。本文主要介绍review。

Phabricator的代码审查功能目前没有图形化的客户端，只能通过命令行进行操作，使用的工具为Arcanist。
