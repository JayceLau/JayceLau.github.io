Oracle RAC 12cR2安装手册(5)--数据库实例的创建

  2017-07-11     Oracle·RAC     Tech·Installation

在node01上执行

创建一个数据库→下一步👆🏻

高级配置→下一步👆🏻

选择自定义数据库→下一步👆🏻

选择所有节点→下一步👆🏻

设置全局数据库名和SID前缀，选择创建空白容器数据库→下一步👆🏻

注：
此处的SID前缀需要与环境变量里的ORACLE_SID对应：
例如，SID前缀设置orcl，ORACLE_SID需要设置为orcl1和orcl2。

选择ASM存储类型并选择DATA磁盘组→下一步👆🏻

启用快速闪回去，并选择DATA磁盘组，大小设置为5120M→下一步👆🏻

确定继续👆🏻

默认→下一步👆🏻

选择自动共享内存管理，并设置Memory Target约为物理内存的80%👆🏻

块大小设置为32768B，进程数设置为1500👆🏻

字符集选择AL32UTF8👆🏻

连接模式选择专用服务器模式→下一步👆🏻

确认继续→下一步👆🏻

不配置EM→下一步👆🏻

使用统一密码→下一步👆🏻

创建数据库→下一步👆🏻

再次确认继续👆🏻

执行先决条件检查👆🏻

忽略全部→下一步👆🏻

确认继续👆🏻

完成👆🏻

创建进行中👆🏻

创建完成→关闭👆🏻
分别在node01和node02上执行lsnrctl status查看实例运行状态👇🏻

node01状态正常👆🏻

node02状态正常👆🏻
执行sqlplus / as sysdba登录数据库👇🏻

登录成功👆🏻
执行show pdbs查看PDB状态👇🏻

可以看到一个种子PDB👆🏻
至此，完成了数据库实例的创建。
