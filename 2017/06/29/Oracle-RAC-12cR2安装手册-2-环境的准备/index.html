Name node01 node02
Public IP 172.16.10.11 172.16.10.12
Private IP 192.168.10.11 192.168.10.12
Virtual IP 172.16.10.21 172.16.10.22
SCAN IP 172.16.10.31 172.16.10.32
172.16.10.33

注：

  1. Public IP, Virtual IP, SCAN IP必须在同一个子网下；
